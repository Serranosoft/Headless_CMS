
# Infoerrores

Infoerrores ha sido creado con [Frontity](https://frontity.org/).

Se trata de un blog Wordpress Headless creado para solventar todo tipo de errores, desde errores en plataformas como Disney, Netflix, Amazon hasta códigos de errores, se le aporta al usuario una solución/guía muy detallada con imágenes

Este proyecto esta orientado al SEO para obtener una suma elevada de usuarios y obtención de ingresos por parte de Adsense.

<img width="250px" align="center" margin="0 auto" src="https://wp.infoerrores.com/wp-content/uploads/2021/03/infoerrores-web.png" />

## Tecnologías usadas

Esta página web ha sido creado con el framework Frontity [Frontity](https://frontity.org/).

Ha sido construida con ReactJS en el frontend y utiliza el dominio [Panel de Admin](https://wp.infoerrores.com/) como REST API Backend para la solicitud de datos a la base de datos.

### Librerías externas

- [Emotion](https://github.com/emotion-js) -> CSS-in-JS para componentes React. Optimizando el rendimiento al solo cargar el CSS necesario cuando el componente se monta.
